# UI Requirements

## Web app landing page (e.g. index.html):

As a starting point, refer to the image titled 'homepage' located in the 'docs' folder.

Element functionality is as follows:

1.  Select existing video from local file system.
3.  Click to store a previously classified video.
4.  Enter text describing the video to generate.
5.  Click to prompt the video generation model.
6.  Click to request additional, similar videos from the generation model.
7.  Click to end the session.
8.  Enter the emotion type viewed in the video.
10. Click to delete the image.
